# Pente-a-kill
Projet workshop EPSI 2017 I4

Reconstitution du jeu Pente avec 2 IA.

## Installation

Installer les dépendances:
`npm install`

Lancer le serveur:
`npm start`

L'application se trouve sur `localhost:3000`
